# Cálculo datas
É um projeto bem básico que fiz há tempos atrás, hoje encontrei o código e estou apenas commitando.

# Pré-requisitos
- Java 8+

# Como Usar
- Rodar a classe principal **FaltaHoras**.
- Colocar no campo **EVENTO** a data desejada.
- Colocar no campo **NOME_EVENTO** o nome do.

Exemplo:
```java
	private static final LocalDateTime EVENTO = LocalDateTime.of(2020, 1, 1, 00, 00);
	private static final String NOME_EVENTO = "ano novo";
```
